## Features

**Supports:** mp3, flac, opus, ogg, wav, aac, m4a, wma, aiff, amr, and webm.

- Reads ID3 tags and displays artist, album, and cover art
- Quickly switch tracks from the playlist
- Search for songs from the search bar
- Use keyboard shortcuts to control playback
- Download audio from YouTube with automatic metadata tagging
- Identify songs using audio fingerprinting (AcoustID + MusicBrainz)
- Generate AI-powered lyrics using Whisper transcription
- Create and manage playlists
- System tray controls for background playback
- Real-time audio visualizer

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Play / Pause | `Space` |
| Next track | `Arrow Right` |
| Previous track | `Arrow Left` |
| Volume up | `Arrow Up` |
| Volume down | `Arrow Down` |
| Mute / Unmute | `M` |

## Installation

```bash
# Clone the repository
git clone https://github.com/Abdullahu5mani/Music-Electron-App.git
cd Music-Electron-App

# Install dependencies
npm install

# Run in development mode
npm run dev
```

### Build

```bash
npm run build
```
